# CORPO : COndominium Repair Plan Optimizer
[![tktIsDQN](https://img.shields.io/badge/tkt-DQN-red.svg)]
## installation
(Developed under `node v 7.1.0`)

` npm install `

## deploy

` node server.js `

## dev

` node run server `

Then access ` localhost:8080/webpack-dev-server/index.html ` on your brower
